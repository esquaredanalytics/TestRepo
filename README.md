# TestRepo

This is an example repository to show the following: 

1) Clone repository 
`git clone git@github.com:esquaredanalytics/TestRepo.git`
2) Checkout main branch 
`git switch main`
3) Create and switch new branch 
`git switch –c update_readme` 
4) Add and commit changes 
`git add README.md` 
`git commit -m “Updated README to …”`
5) Check out differences 
`git diff README.md`
6) Push changes to remote 
`git push origin`
7) Create pull request
8) Merge pull request

Making a change!
